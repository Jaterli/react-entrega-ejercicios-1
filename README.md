# 🧠 React - Entrega de Ejercicios (Módulo React)

[![React](https://img.shields.io/badge/React-18.2.0-61DAFB?logo=react&logoColor=white)](https://reactjs.org/)
[![React Router](https://img.shields.io/badge/React_Router-6.8.0-CA4245?logo=react-router&logoColor=white)](https://reactrouter.com/)
[![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript&logoColor=black)](https://developer.mozilla.org/es/docs/Web/JavaScript)

Proyecto que agrupa **9 ejercicios prácticos** desarrollados con **React** como parte de la **Entrega 1 del módulo de React** del *Máster de Desarrollo Web Full Stack*.

Cada ejercicio implementa un componente funcional independiente, utilizando **Hooks** (`useState`, `useEffect`, `useRef`) y buenas prácticas como accesibilidad (`aria-*`), diseño responsive y rutas con `React Router`.

---

## 📦 Ejercicios incluidos

| # | Ejercicio | Descripción |
|---|-----------|-------------|
| 1 | 🎨 **Cambiador de Color** | Cambia el color de fondo del body de forma aleatoria. |
| 2 | 🔢 **Contador de Clics** | Incrementa un contador y muestra mensajes motivadores aleatorios. |
| 3 | 📝 **Lista Dinámica** | Agrega y elimina elementos de una lista en tiempo real. |
| 4 | 🔍 **Búsqueda en Tiempo Real** | Filtra elementos de una lista mientras escribes. |
| 5 | 🧮 **Calculadora Sencilla** | Realiza operaciones básicas con validación de errores. |
| 6 | ⏱️ **Temporizador** | Control de tiempo con iniciar, pausar y reiniciar. |
| 7 | 🔐 **Generador de Contraseñas** | Crea contraseñas seguras con longitud personalizable. |
| 8 | 📊 **Contador de Palabras y Caracteres** | Analiza texto en tiempo real (sin espacios). |
| 9 | ✅ **Lista de Tareas con LocalStorage** | CRUD de tareas con persistencia local. |

## 🚀 Tecnologías utilizadas

- ⚛️ **React 18** (componentes funcionales + Hooks)
- 🧭 **React Router DOM** (navegación entre ejercicios)
- 🎨 **CSS3** (Mobile First, Grid, Flexbox, diseño responsive)
- ♿ **Accesibilidad** (atributos `aria-*`, roles, `aria-live`)
- 💾 **LocalStorage** (persistencia en el ejercicio 9)
- 🔐 **Criptografía segura** (`crypto.getRandomValues` en ejercicio 7)

## 🛠️ Instalación y ejecución

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/react-entrega-ejercicios.git
cd react-entrega-ejercicios
```

2. Instala las dependencias:
```bash
npm install
```

3. Ejecuta el proyecto en modo desarrollo:
```bash
npm run dev
```

4. Abre [http://localhost:5173](http://localhost:5173) en tu navegador.
